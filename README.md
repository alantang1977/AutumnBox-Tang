# AutumnBox
# 秋之盒：天猫魔盒M13-云OS6.1.0系统精简去广告方法


## 开始破解
1）电脑上运行秋之盒工具箱，电脑和盒子在同一局域网

2）系统设置里恢复出厂设置，恢复后链接无线网

3）U盘插入盒子，安装冰箱app，打开系统设置的远程调试

4）在电脑上用秋之盒工具箱链接魔盒m13

5）通过秋之盒激活冰箱

6）盒子上打开冰箱冻结如下应用
